NB!!!! Man kan alltid gå tilbake til normal mode ved å trykke esc, og insert mode med enten i eller a
# Oppgave 1: Bruk 'x' for å slette bokstaven 'x'
Instruksjoner:
- Bruk 'h' for å bevege cursoren til venstre
- Bruk 'l' for å bevege cursoren til høyre
- Bruk 'j' for å bevege cursoren til nedover
- Bruk 'k' for å bevege cursoren til oppover
- Naviger til 'x' ved å trykke 'l' til cursoren er over 'x'
- Trykk 'x' for å slette bokstaven
- NB denne lagres til clipboard så om du deretter trykker p vil du lime inn den slettede bokstaven!

hello world

# Oppgave 2: Bruk 'dd' for å slette hele linjen
Instruksjoner:
- Bruk 'j' for å bevege cursoren ned en linje
- Bruk 'k' for å bevege cursoren opp en linje
- Naviger til linjen "denne linjen skal slettes" (cursoren kan være hvor som helst på linjen)
- Trykk 'dd' for å slette hele linjen
denne linjen skal slettes
behold denne linjen

# Oppgave 3: Bruk 'dw' for å slette ordet 'slett'
Instruksjoner:
- Bruk 'h' for å bevege cursoren til venstre
- Bruk 'l' for å bevege cursoren til høyre
- Bruk 'j' for å bevege cursoren ned
- Bruk 'k' for å bevege cursoren opp
- Naviger til starten av ordet 'slett' ved å bruke hjkl
- Når cursoren er på 's' i 'slett', trykk 'dw' for å slette ordet

jeg vil slett beholde resten

Tips for effektiv navigering:
- 'h' er venstre (tenk på at h er lengst til venstre av hjkl)
- 'j' er ned (tenk på j som en pil som peker ned)
- 'k' er opp
- 'l' er høyre
- Du kan holde inne en navigasjonstast for å bevege cursoren raskere
